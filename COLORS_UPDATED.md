# TOPIA Color System - Updated

Your complete TOPIA color palette with the new bright yellow-green!

## Updated Color Palette

### Base Colors
| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| **Red-Orange** | `#FF5C34` | rgb(255, 92, 52) | Primary actions, CTAs |
| **Bright Yellow-Green** | `#e4fe52` | rgb(228, 254, 82) | Highlights, attention, energy |
| **Near Black** | `#171718` | rgb(23, 23, 24) | Dark backgrounds, text |
| **Off White** | `#FEFEFF` | rgb(254, 254, 255) | Light backgrounds |

### Support Colors
| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| **Electric Blue** | `#4F46FF` | rgb(79, 70, 255) | Links, interactive elements |
| **Neon Green** | `#00FF88` | rgb(0, 255, 136) | Success states, positive actions |
| **Hot Pink** | `#FF5BD7` | rgb(255, 91, 215) | Accents, creative touches |

## What Changed

The acid yellow has been updated to **#e4fe52** - a brighter, more electric yellow-green that really pops! This new shade:
- Has more green undertones for extra vibrancy
- Is even brighter and more eye-catching
- Perfect for the TOPIA brand energy
- Works beautifully with the neon green (#00FF88)

## Usage in Components

All existing usage remains the same - just use the Tailwind classes:

```tsx
// Yellow is now the bright yellow-green
<div className="text-yellow">Bright yellow-green text</div>
<div className="bg-yellow">Bright yellow-green background</div>
<div className="border-yellow">Bright yellow-green border</div>
```
